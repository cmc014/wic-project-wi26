This project is a Zodiac-based Pop Culture Recommendation Calculator. 
The user will enter their birthday and preferred genre to get four unique Pop Culture Recommendations.  

Features: 
- Zodiac Calculator with Interactive Dropdown
- Zodiac Information Card Deck
- Sidebar

How to Run: Open index.html with Live Server in VSCode

Technologies Used: 
- JavaScript
- HTML
- CSS
- Github (documentation)

Authors: An Nghi Nguyen, Sarah Kim, Elysia Jin, Christina Asinobi, Callie Chen
